About This Project

This project analyzes the PetFinder.my Adoption Prediction dataset from Kaggle. The goal is to understand what factors influence how quickly a pet gets adopted — from tabular features like age, breed, vaccination status, and health condition, to image-level attributes like photo quality, brightness, resolution, and the number of photos per listing.

The workspace also includes a Text EDA pipeline for the News Category dataset, covering schema inspection, class distribution, missing values, text length statistics, timeline shifts, and dominant content terms.
