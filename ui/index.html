About This Project

The objective of this project is to perform Exploratory Data Analysis (EDA) on datasets of different modalities so students can practice understanding data before building downstream AI and data science solutions. The assignment requires coverage of tabular, text, and image data, with one dataset analyzed for each modality.

For each report, the work follows the core EDA tasks stated in the brief: inspect dataset size and schema, review sample records, compute basic statistics such as distributions, missing values, and duplicates, create meaningful visualizations, and summarize notable findings or potential issues such as imbalance, noise, or preprocessing needs.
