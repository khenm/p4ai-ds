Feature Selection Strategy

During the Exploratory Data Analysis phase, we evaluated the interactions between all variables via Cramér's V and numerical Correlation Matrices. We empirically proved that there is zero redundancy between the attributes (all inter-feature correlation scores were extremely low, practically establishing pure independence).

Consequently, our Feature Selection strategy deliberately retains all 8 predictor features to train the algorithms, preventing any loss of variance matrix dimensionality:

  • Numerical Predictors: experience_years, skills_count
  • Categorical Nominative: job_title, industry, location, remote_work
  • Categorical Ordinal: education_level, company_size

Algorithm Performance Benchmark

Insights (Proving the "Exponential Multiplier")

Standard Linear Regression serves as a robust baseline but underperforms severely when estimating the long tail of top earners. XGBoost and LightGBM successfully map out non-linear exponential interactions (such as having an Advanced Degree inside an Enterprise), leading to significantly higher R² Accuracy and massively reduced Root Mean Square Error (RMSE).
